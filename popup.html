How to Use

Single Archive

Go to any Gemini, ChatGPT, or Claude chat URL and keep the input empty. Click Archive to save the current tab.

Batch Archive

Paste multiple shared conversation URLs (one per line) into the text box and click Archive. The extension will open, scrape, and save them sequentially.
